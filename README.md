Instructions to install and work with Hugo blogs in WIndows Enviroment

1) Open PowerShell as Administrator
2) Run following commands from c:\windows\system32
     1) winget install Hugo.Hugo.Extended
     2) winget install Git.Git
     3) winget install GoLang.Go
4) Go to c:\
5) mkdir websites
6) cd C:\websites
7) run `hugo new site mysite --format="yaml"`
8) run dir
9) cd mysite
10) dir
11) cd themes
12) git clone https://github.com/gurusabarish/hugo-profile
13) dir
14) You will see hugo-profile folder
